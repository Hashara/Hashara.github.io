Healers

Semester 3 Project
Mar 2019 - June 2019

Description

A web application to connect students and mentors in the university. It eases communication between students and mentors within the university, the mentors could keep track of his/her student and could assign tasks.

Used technologies

  • PHP
  • MySQL
  • Html
  • CSS

My Role

I developed the authentication system of the application, messaging function, mentor selecting function both backend and the frontend part. This was the first web-based application I built. We did not allow to use of frameworks to develop this application. We develop this application with MVC architecture. I was able to gain a lot of experience and learned the basics of web development.
