SmartEdkit

Project for code with WIE competition - 2019
Jul 2019 - Aug 2019

Description

A web application for school administrators and a mobile application for school teachers, intended to improve the efficiency of the school system. This system would help to monitor the attendance of both teachers and the students, teachers could take their notes, teachers would receive the notifications reminding the next period.


Used technologies

  • Flutter
  • Angular JS
  • Firebase

My Role

I developed the mobile application of teachers using flutter.This was a minimum viable project build for Code with WIE competition and this was my first mobile application. By developing this mobile application I was able to get experience in mobile application development.
