Multi-Model Radiography Analysis Framework for Pneumonia and COVID-19

Research and Development Project
Ongoing

Description

A chest x-ray analysis framework can be used for Pneumonia and COVID-19 diagnosis

Used technologies

  • Python
  • Machine Learning
  • Convolutional Neural Network
  • Tensorflow
  • Computer Vision

Progress so far

We have used four existing architectures (MobileNetV2, InceptionV3, Xception, Resnet50) with adding top layers and we were able to gain high accuracy, precision, recall, and F1 score.
