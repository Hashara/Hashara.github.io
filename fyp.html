NCMS - WAKANDA

SparkX | Individual Assignment
June 2021 - July 2021

Description

A web application for COVID management in Wakanda. I developed the REST API by using Java servlet and designed the PostgreSQL database for the system. For the authentication system, I have used JSON Web Token (JWT), and for password hashing used jbcrypt. For logging functionality, I have used Log4j. For the frontend, I have used React with React-redux. This was the first project that I have used React-redux.

Used technologies

  • Java
  • Java Servlet
  • PostgreSQL
  • React
  • React-redux
