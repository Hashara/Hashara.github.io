Distributed chat system

In this project, a web-based system and two mobile applications, with REST API as the backend were developed where the passengers can reserve bus seats online by paying the amount. This will facilitate passengers and conductors, making the ticketing process more convenient. A mobile application will be developed for the driver/conductor to handle the ticketing process from their side. The web application will facilitate both bus owners and passengers. After paying the relevant amount through the web app or the mobile app the passenger should receive a receipt confirming the reservation and the conductor/driver should receive the information about the reservations.
