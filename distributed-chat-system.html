Distributed chat system

Semester 7 | Distributed System Project
Sep 2021 - Oct 2021

Description

This project is a distributed chat application. The project's goal is to create chat servers where clients can connect and chat with one another. Chat servers offer clients to connect with unique ids, create/join/delete chat rooms, chatting with other clients etc. For leader election, we have used bully algorithm, for failure detection we have used heartbeat. We have used the Netty framework for TCP connection.

Used technologies

  • Java

My Role

I developed client requests for deleting room, room list, moving from room and joining for another room. Also done part of the bully algorithm.
