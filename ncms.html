NCMS - WAKANDA

SparkX | Individual Assignment
June 2021 - July 2021

Description

A COVID management tool that has authentication, bed reservation, COVID statistic review. This applications' REST API was built using JAVA Servlet, the UI was built using React with React-redux, and PostgreSQL was used as the database. A web application for COVID management in Wakanda. Additionally for the authentication system was developed using JSON Web Token (JWT), and for password hashing used jbcrypt. For logging functionality, Log4j has been used.

Used technologies

  • Java
  • Java Servlet
  • PostgreSQL
  • React
  • React-redux
